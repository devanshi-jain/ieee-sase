    • d3, a JavaScript library for manipulating documents based on data

    • q, tool for making and composing asynchronous promises in JavaScript
    • arbor.js, a graph visualization library
    • typeahead.js, autocomplete search functionality from Twitter
    • KeyboardJS, library for binding keyboard combinations
    • 30 Seconds of code, a utility library with optimized functions
    • Mousetrap, another library for binding keyboard combinations
    • Quill, a rich text editor
    • Numeral.js, a library for formatting and manipulating numbers
    • interact.js, JavaScript drag and drop, resizing and multi-touch gestures with inertia and snapping for modern browsers
    • Sortable, minimalist JavaScript library for modern browsers and touch devices that makes manipulating lists so easy
    • RequireJS, an asynchronous file and module loader
    • Underscore, a utility library
    • Lo-Dash, for those who want more from underscore
    • AmplifyJS, solves a set of problems (such as cross browser local storage)
    • LeafletJS, an Open-Source JavaScript Library for Mobile-Friendly Interactive Maps
